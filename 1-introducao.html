Variáveis

  • Numéricos: int, float, double
  • alfanumérico: char
  • void
  • ponteiros
Além dos tipos simples há tipos compostos como: arranjos, registros, enumerações e arquivos.

Variáveis

Tipo Espaço de memória Números armazenados
int 4 bytes -214783648 a 214783647
float 4 bytes -3.4e38 a +3.4e38
double 8 bytes -1.7e308 a +1.7e308
char 1 byte caracteres do código ASCII

Entrada de dados


	scanf("%d", &idade);
	scanf("%f", &salario);
	scanf(" %c", &sexo);
Observe o espaço em branco na entrada de dados para uma variável do tipo char

Saída de dados


	printf("A idade do aluno é %d", idade);
	printf("\nO salário do funcionário é de R$ %f", salario);
	printf("\nO salário do funcionário é de R$ %6.2f", salario);
	printf("\nO sexo informador é: %c", sexo);
Você pode formatar um número com casas decimais!
